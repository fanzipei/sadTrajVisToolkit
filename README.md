# sadTrajVisToolkit
Toolkits for trajectory visualization

## installation
#### 0. install dependencies

folium >= 0.7

conda install folium -c conda-forge

#### 1. clone the project

git clone git@github.com:sadsadcool/sadTrajVisToolkit.git

#### 2. install the tool
cd sadTrajVisToolkit

pip install .
